### How you init on terminal:

_1 Iniciar o node_
`npm init -y`

_2 Instalar ferramentas do backend_
`npm install cors dotenv express mysql2`

_3 Instalar um live server_
`npm install --save-dev nodemon`

_4 Abra o package.json e crie a linha no scripts:_
`"dev": "nodemon app.js",`

_5 Comando para iniciar o backend no terminal:_
`npm run dev`

_6 Se a pasta node_modules nao existir rode:_
`npm install`
